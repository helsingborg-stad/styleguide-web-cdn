Grid

HbgPrime's grid system is a 12 column responsive grid.
ATTENTION: Remove the .grid-example class before using.

Example

grid-md-3
grid-md-9
<div class="grid grid-example">
    <div class="grid-md-3">grid-md-3</div>
    <div class="grid-md-9">grid-md-9</div>
</div>
